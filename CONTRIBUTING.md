### Contribution
To contribute do this:  
First you will need node.  
fork the repo and create a new branch, run  
`npm install` to install the dependencies and make your edits  
then once done, check all tests  
do check the tests run `./install.sh`    
and run `noodle ./test_suite.bowl` (while cd'd in the test directory!)  
then issue a pull request, and i'll review as soon as possible
